# Asistente Virtual con Python

Este repositorio contiene el código fuente de un Asistente Virtual desarrollado en Python.

## Descripción

Este Asistente Virtual es capaz de realizar varias tareas, como abrir programas, buscar información en la web, reproducir música, entre otras funcionalidades. Fue creado con el objetivo de facilitar la interacción con el sistema operativo y mejorar la productividad del usuario.

## Instalación

Para instalar y ejecutar este asistente virtual, sigue estos pasos:

1. Clona el repositorio en tu máquina local.
2. Instala las dependencias necesarias con el comando `pip install -r requirements.txt`.
3. Ejecuta el script principal con el comando `python main.py`.

## Requisitos y Uso

Para poder utilizarlo de forma correcta necesitas una ApiKey de Openai y de Microsoft Azure. Este ultimo se puede omitir usando otra libreria de reconocimiento de voz y de texto a voz.

Para interactuar con el asistente, simplemente habla después de iniciar el programa. El asistente está diseñado para reconocer varias instrucciones. Por ejemplo, puedes decir "abrir Google" y el asistente abrirá Google en tu navegador predeterminado.

## Contribuciones y Notas.
